This session covers how to add authentication to MCP servers in Python. It explores three approaches to restricting access: private networks, key-based access, and OAuth-based access. The bulk of the session focuses on OAuth 2.1 flows for MCP including PRM, CIMD, DCR, and integration demos with Keycloak and Microsoft Entra.
